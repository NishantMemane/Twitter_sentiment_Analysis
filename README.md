# 🧠 Twitter Sentiment Analysis using CNN

A Deep Learning project that classifies live tweets into **Positive**, **Negative**, or **Neutral** sentiments using a **Convolutional Neural Network (CNN)** trained on the **Sentiment140 dataset**.

## 🚀 Features
- CNN-based sentiment analysis
- Real-time tweet fetching with `snscrape`
- Graphical sentiment visualization
- Frontend (Streamlit/Flask) for user interaction

## 📦 Requirements
```bash
pip install tensorflow pandas scikit-learn snscrape
